#1

First, remindr will send an invitation with the photo, title, description, and event time. This initial text will ask whether or not the remindee would like to opt in to receiving remindrs for this event.

#2

We want to make sure that remindrs are only being sent to the people who want to receive them. With remindr's opt-in process, we can make this happen.

#3

As soon as the remindee opts in, they are given a automatic confirmation text so that they know they have opted in.

#4

Finally, the remindee can receive remindrs at the specified alert times.
